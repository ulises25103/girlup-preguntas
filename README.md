# girlup-preguntas

Cuestionario web **"¿Qué intersección del feminismo te representa más?"** para Girl Up Argentina.

Sitio estático de una sola página: el usuario responde 3 preguntas y se le muestra cuál es su "eje"
(deporte, derechos digitales, cuidados, incidencia política, educación, justicia menstrual o
comunicación), según la categoría que más puntos haya sumado.

## Cómo verlo
Abrí `index.html` en cualquier navegador, o desplegalo en Vercel (no requiere build).

## Archivos
- `index.html` — toda la página (HTML + CSS + JS en un solo archivo).
- `girl-up-logo-blanco.png` — logo usado en la portada.
